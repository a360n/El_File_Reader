# EcoLAB EL File Reader & AI Dataset Auditor (C# .NET)

أداة سطح مكتب وتدقيق مبنية بلغة **C# وإطار عمل .NET** لإعادة هيكلة وتدقيق بيانات ألواح **EcoLAB** الشمسية من شركة **EcoProgetti**، وتجهيز بيانات نقية ومضمونة لتدريب نماذج الذكاء الاصطناعي.

---

## 🚀 المميزات الأساسية (Core Features)

1. **إعادة الهيكلة والتجميع التلقائي (Automatic File Restructuring)**:
   - ينشئ المجلدات الرئيسية `container` و `Restructured`.
   - يقوم بفرز وتجميع الثلاثيات المكتملة بناءً على المعرف المشترك:
     - الصورة الخام: `XXXXX.1.tif` ⬅️ تُسمى `row.tif`
     - ملف البيانات: `XXXXX.el` ⬅️ يُسمى `info.el`
     - الصورة المؤشرة: `ANM..._XXXXX-1.tif` ⬅️ تُسمى `marked.tif`
   - يُنشئ مجلداً فرعياً لكل لوح يحمل تاريخ إنشاء ملف الـ EL بالشكل `YYYY-MM-DD_HH-mm-ss`.
   - يتم حذف الملفات بعد الفرز من `container` وتجاهل الملفات غير المكتملة.

2. **شاشة التدقيق والمقارنة (Interactive Audit Screen)**:
   - عرض صورتي `row.tif` و `marked.tif` جنباً إلى جنب مع إمكانية المعاينة والتكبير.
   - قراءة وعرض تفاصيل `info.el` أسفل الصور مع المعايرة التلقائية (A01 - F24).
   - **الزر الأول (🟩)**: `معلومات info مطابقة لل marked`
   - **الزر الثاني (🟥)**: `معلومات info غير مطابقة لل marked` (يفتح لوحة اختيار الأعمدة A-F والصفوف 01-24 مع إمكانية إضافة حتى 10 عيوب).
   - **الزر الثالث (💾)**: `حفظ وإقرار` (يحفظ القرار وينتقل تلقائياً للوح التالي).
   - زر عودة للوح السابق + زر الانتقال لصفحة السجل.

3. **صفحة سجل المطابقة والتقارير (Audit Log & Reports Page)**:
   - جدول تفصيلي يوضح حالة الألواح وتعديلات المصحح.
   - كارت معلومات وإحصائيات نحسب فيه: (إجمالي الألواح، الألواح المطابقة، الألواح غير المطابقة، **نسبة دقة قارئ الـ EL**).
   - **توليد تقرير PDF**: تصدير التقرير كاملاً بملف جاهز للطباعة والمشاركة.

## 🛠️ التشغيل على نظام Windows

### المتطلبات:
- تثبيت [.NET 8.0 SDK / .NET 9.0 SDK](https://dotnet.microsoft.com/download)

### طريقة التشغيل:
1. افتح موجه الأوامر (CMD / PowerShell) في مجلد المشروع:
   ```cmd
   cd EcoLabReaderApp
   dotnet run
   ```
2. سيفتح التطبيق تلقائياً في متصفحك المفضل على المسار:
   `http://localhost:5199`

## 📂 هيكلية المجلدات عند العمل:

```text
El_File_Reader/
├── container/              <-- اضع الملفات غير المرتبة هنا
├── Restructured/           <-- المجلد الذي يحتوي الألواح المنظمة بعد الفرز
│   ├── 2026-04-01_16-01-05/
│   │   ├── row.tif
│   │   ├── info.el
│   │   └── marked.tif
│   └── audit_log.json      <-- سجل القرارات المحفوظة
```
